# N7 Banking Landing Page

A modern and responsive banking landing page built using Next.js, TypeScript, and Tailwind CSS.

## Tech Stack

* Next.js
* TypeScript
* Tailwind CSS

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/vasavi-22/n7-frontend.git
```

### 2. Navigate to the project directory

```bash
cd n7-frontend
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start the development server

```bash
npm run dev
```

The application will be available at:

```bash
http://localhost:3000
```

## Build for Production

```bash
npm run build
```

## Run Production Build

```bash
npm start
```

## Features

* Fully responsive UI
* Modern banking landing page design
* Reusable UI components
* Optimized fonts using `next/font`
* Tailwind-based styling system
